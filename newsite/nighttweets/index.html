#nighttweets

#nightTweets is a platform that allows me to tweet in my sleep. This project explores the concept of outsourcing internet activity to your sleeping self in order to accommodate the urgency we feel to share on social media.

The program analyzes my twitter account to create a vocabulary to use when it creates new tweets or messages. Every five minutes, it reads Zeo data and takes a different action on twitter relevant to which stage of sleep I'm experiencing: light sleep, deep sleep, or REM.

#nightTweets runs in Processing using the Twitter4J + Rita libraries, and uses data from the Zeo sleep tracker.
